# CogniTrust v2.1

CogniTrust is an open-source AI confidence evaluator. It applies confidence thresholds to AI-generated responses and displays corresponding color-coded indicators.  

## Features
- Four-level confidence system: green (>=98%), yellow (80-97%), orange (60-79%), red (<60%)
- Differentiates between empirical and contextual facts
- Interactive demo included
- Clickable placeholder source links

## Installation
```bash
git clone <your_repo_url>
cd cognitrust
python3 main.py
